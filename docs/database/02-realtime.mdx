In this chapter, we will share with you some of the **Open Source Database(OSDB) on GitHub in real time** measured by different metrics including the number of stars, PRs, contributors, countries, regions and so on.

Note:
1. The real-time analytical results are updated hourly.
2. You can move your cursor onto any of the repository bars/lines on the chart and get the exact number.
3. The SQL commands above each chart are what we use on our TiDB Cloud to get the analytical results. Try those SQL commands by yourselves on TiDB Cloud with this [10-minute tutorial](/blog/try-it-yourself/).
